<h1>אני הדל ועזי עיר ומגדל</h1>
שלום שבזי
<p>חתימה: אלשבזי משתא (שלם) – מאה יז</p>
<blockquote><p>אֲנִי הַדַּל / וְעָזִּי עִיר וּמִגְדַּל / וּמִבְטַחִי וְחֶשְׁקִי שֵׁםאֲדֹנָי</p></blockquote>
<blockquote><p>לְחֵן שַׁדַּי / אֲנִי נִכְסָף בְּלִי דַּי / בְּאַהְבָתִי וְשֵׂכֶלרַעֲיוֹנָי</p></blockquote>
<blockquote><p>שְׁבִיב נוּרָא / דְּמַלְכָּא אֶתְקְטִירָא / בְּתוֹךְ לִבִּי לְהָסִיר אֶתיְגוֹנָי</p></blockquote>
<blockquote><p>בְּסוֹד אַהְבַת / הֲדָסָה הַמְּכַסָּה / עֲלֵי רֹאשִׁי בְּשָׁעַתהֶגְיוֹנָי</p></blockquote>
<blockquote><p>זְכוּת עוֹרֵר / יְדִיד עַל דַּל מְשׁוֹרֵר / בְּשִׁיר חָדָשׁ לְשַׂמֵּחַהֲמוֹנָי</p></blockquote>
<blockquote><p>יְבִיאוּנִי / בְּחֵן חַסְדֵי אֲדֹנָי / לְהַר קָדְשׁוֹ בְּיוֹם בּוֹארֹאשׁ גְּאוֹנָי</p></blockquote>
<blockquote><p>מְחַכֶּה קוֹל / תְּרוּעָה יוֹם יְשׁוּעָה / וְשִׁבְטֵי יָהּ וְדִגְלֵיהֶםלְפָנָי</p></blockquote>
<blockquote><p>שְׁאוֹן גַּלֵּי / זְמָן אֶשְׁכַּח בְּתֵימָן / וְדוֹד נֶאְמָן / יְכַסֶּהכָּל עֲוֹנָי</p></blockquote>
<blockquote><p>תְּנִי הַלֵּל / יְחִידָתִי חֲצוֹת לֵיל / לְצוּר גּוֹלֵל / לְאוֹר לִבִּיוְעֵינָי</p></blockquote>
<blockquote><p>אֲדוֹן הַכֹּל / וְשָׂם דַּעַת וְהַשְׂכֵּל / בְּפִי רַחְשִׁי וְגַם שֵׁמַעבְּאָזְנָי</p></blockquote>
<blockquote><p>וְשִׁין לַמַדּ / וּמִים בִּשְׁמִי יְלַמֵּד / זְכוּת עָלַי וְעַל חֶבְרַתזְקֵנָי.</p></blockquote>
