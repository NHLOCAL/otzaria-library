<h1>[כְּבָר נוֹאַשׁ]</h1>
שלמה אבן גבירול
<blockquote><p>כְּבָר נוֹאַשׁ לְבָבִי מִמְּצֹא אִי / בְּאַנְשֵׁי הַזְּמַן וּבְיוֹשְׁבֵיאִי</p></blockquote>
<blockquote><p>וְאֵיךְ לִי תֹאמְרוּ: הִנֵּה וְהִנֵּה! / יְעִירֵנִי לְבָבִי, אִי לְךָ,אִי!</p></blockquote>
<blockquote><p>אֲשֶׁר אֶכְסֹף לְאִישׁ שֶׂכֶל וּבִינָה / וְאֵין בַּמָּה אֲרַוֶּה אֶתצְמָאִי</p></blockquote>
<blockquote><p>וְלֹא אֶפְחַד בְּבוֹאִי אֶל אֲדָמָה / לְמַעַן כִּי יְדַעְתִּיהָבְּבוֹאִי</p></blockquote>
<blockquote><p>וְאֵדַע מִזְּמַנִּי אַחֲרִיתוֹ / כְּאִ[לּוּ בֶּן] בְּעוֹר הָיָהנְבִיאִי</p></blockquote>
<blockquote><p>וְהָיִיתִי תְמוֹל מֵרֹאשׁ יְרֵאָיו – / וְהִשְׁכִּים הַזְּמַן הַיּוֹםיְרֵאִי</p></blockquote>
<blockquote><p>וְגָבַרְתִּי בְּנוֹת יָמִים, כְּאִלּוּ / אֲצִילֵי יִשְׂ[רְאֵלִים] עַלצְבָאִי</p></blockquote>
<blockquote><p>וְאַל יֹאמַר גְּבַהּ לֵב כִּי אֲנִי הוֹד / סְפָרַד עַד יְהוֹדֵנִימְשַׂנְאִי</p></blockquote>
<blockquote><p>וְכֵן אַל יִשְׁמְעוּ א[וֹ]תִי יְרִיבַי / עֲדֵי כִי אֶתְּנֵם אוֹתִיוּפִלְאִי</p></blockquote>
<blockquote><p>הֲכִי לֹא שֶׁאֱלַת נַפְשִׁי אֲמַלֵּא / עֲדֵי תִֽהְיֶה תֵבֵל מְלֹאִי</p></blockquote>
<blockquote><p>וְלֹא אֶשְׁקֹט כְּמוֹ לֹא אֶשְׁקְטָה [עַד / אֲנַשֵּׂא אֶת] יְקוּתִיאֵלנְשִׂיאִי</p></blockquote>
<blockquote><p>מְצָאתִיהוּ אֲחִי דַעַת וְהַשְׂכֵּל / וְשָׂמַחְתִּי עֲלֵי שִׂכְלִיבְּמָצְאִי</p></blockquote>
<blockquote><p>[וְאָמְנָם אֵינְךָ שָׂעִיר כְּאָמְרָם / אֲבָל שָׂעִיר עֲלֵי עֶשְׂבִּיוְדִשְׁאִי]</p></blockquote>
<blockquote><p>וְשֶׁמֶשׁ מִמְּךָ תִזְרַח בְּתֵבֵל / וְתֹאמַר: עִמְּךָ הָיָה מְבוֹאִי.</p></blockquote>
<blockquote><p>יְהִי לָעַד שְׁמָךְ הַטּוֹב מְרוֹמָם / וְלָעַד יַעֲלֶה שִׂיאוֹנְשִׂיאִי.</p><p>שורות נוספות בגרסה אחרת:</p></blockquote>
<blockquote><p>אֲשֶׁר עֵת אֶקְרְאָה אֵלָיו יְמָאֵן / וְלֹא יִיגַע וְיָגַעְתִּיבְקָרְאִי</p></blockquote>
<blockquote><p>וְדִמִּיתִי הֱיוֹת יִשְׁעִי וְסַלְעִי / וְגִיל עִלְזִי וְצוּר עֻזִּיוְרַבְּאִי</p></blockquote>
<blockquote><p>נְתַתִּיהוּ לְמוּל חַיַּי וְשַׂמְתִּיו / כְּאֵל נֹכַח יְמֵי מִדַּתצְבָאִי</p></blockquote>
<blockquote><p>וְלָמָה אַחֲרַי תִּהְיֶה וְתִסֹּג / וְתַשְׁחֶה נֵס תְּהִלָּתְךָבְּנָשְׂאִי.</p></blockquote>
