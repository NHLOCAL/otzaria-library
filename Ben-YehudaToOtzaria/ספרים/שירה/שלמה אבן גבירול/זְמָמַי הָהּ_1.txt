<h1>זְמָמַי הָהּ</h1>
שלמה אבן גבירול
<blockquote><p>זְמָמַי, הָהּ, בְּהַר כֶּסֶל נְבוּכִים, / וְעֵינַי - יָרְטוּ נֶגְדָּםדְּרָכִים</p></blockquote>
<blockquote><p>וּמַה יִּתְרוֹן בְּאוֹר שֶׁמֶשׁ בְּעֵינַי - / וְלִבִּי יַהֲלֹךְ נֶגְדּוֹחֲשֵׁכִים!</p></blockquote>
<blockquote><p>וְאֶדְרֹךְ עֹז בְּנֵי יָמִים בְּנַפְשִׁי - / וּבָנַי בַּחֲמַת יָמִיםדְּרוּכִים</p></blockquote>
<blockquote><p>וְנִפְתָּה, כַּאֲרוּס תֵּבֵל, לְבָבִי / וְשָׁדֶיהָ בְּיַד זָרִיםמְעוּכִים</p></blockquote>
<blockquote><p>וְרַב לִי עוֹד רְדֹף הַבְלֵי זְמַנִּים / וְהָלוֹךְ אַחֲרֵי לִבּוֹתהֲפוּכִים</p></blockquote>
<blockquote><p>וְהִתְבּוֹסֵס בְּדַם נֹעַר - וּמֵאָז / דְּמֵי נֹעַר בְּיַד יָמִיםשְׁפוּכִים</p></blockquote>
<blockquote><p>וְאִם צָמֵא לְמֵי חָכְמָה וְהוֹמֶה - / וְיָמָיו בַּעֲבוֹת עָמָלמְשׁוּכִים</p></blockquote>
<blockquote><p>רְדוּף יָגוֹן, שְׂבַע קָלוֹן, וּמִדַּם / לְבָבוֹ יִסְּכוּ עֵינָיונְסָכִים</p></blockquote>
<blockquote><p>הֲדָפוּהוּ תְּלָאוֹת מִנְּעוּרָיו / וְחִצֵּיהֶן בְּצַלְעוֹתָיו מְעוּכִים</p></blockquote>
<blockquote><p>וְרַעְיוֹנָיו יְעִירוּהוּ וּבִנְאוֹת / מְצוּקִים יֵלְכוּ הָלוֹךְוּבוֹכִים</p></blockquote>
<blockquote><p>וְיִיגַע לַעֲצֹר בָּהֶם - וְיַמֵּי / מְזִמָּתָם בְּדַלְתֵי אַף מְסִיכִים</p></blockquote>
<blockquote><p>וְיַעְפִּיל לַעֲלוֹת אֶל הַר תְּבוּנָה - / וְרַגְלָיו מִנְּטוֹת אֵלָיוחֲשׂוּכִים</p></blockquote>
<blockquote><p>וּמִי יוֹבִיל לְמִבְצָר יוֹרְדֵי בוֹר / וְיָבִיא בֵית עֲשִׁירֵי בִיןהֲלָכִים</p></blockquote>
<blockquote><p>וּמִילָדִים כְּגִילָם הֵם יְרֵאִים - / וְאֵיךְ יִתְיַצְּבוּ לִפְנֵימְלָכִים</p></blockquote>
<blockquote><p>וּבִנְטוֹת אָהֳלֵי בִין - עַל יְמִינָם / לְשִׂטְנָם נוֹגְשֵׂי סִכְלוּתעֲרוּכִים</p></blockquote>
<blockquote><p>וּמֵהַטּוֹת יְרִיעוֹתָם חֲשׂוּכוֹת - / וְהֵמָּה מֵיתְרֵיהֶם מַאֲרִיכִים</p></blockquote>
<blockquote><p>וְיַחַד יִקְרְאוּ כֻלָּם בְּשֵׁם אֵל - / וּפִסְלֵיהֶם בְּבֵית כֶּסֶלנְסוּכִים</p></blockquote>
<blockquote><p>וְאֶדַּדֵּם לְאַט עַד בֵּית אֱלֹהִים / וְאוֹרֵם לַהֲלֹךְ בִּמְרוֹםהֲלִיכִים</p></blockquote>
<blockquote><p>הֲלִיכִים נוֹסְדוּ עַל קַו אֱמוּנָה / בְּחֻקֵּי אֵל נְתִיבֵיהֶםסְמוּכִים</p></blockquote>
<blockquote><p>וְיֶהְמוּן לַהֲגוֹת בָּהֶם - וְלִנְוֵה / תְעוּדָה מִקְּצוֹתָם נֶהֱלָכִים</p></blockquote>
<blockquote><p>לְאוֹר עוֹלָם אֲשֶׁר הֵאִיר מְכוֹנָהּ / וּבִימִינוֹ אֱמוּנֶיהָתְּמוּכִים</p></blockquote>
<blockquote><p>וְעָלָה מַעֲלוֹת חָכְמָה וְגָבַהּ / וְלִקְרַאת לוֹחֲמָיו יָרִיקחֲנִיכִים</p></blockquote>
<blockquote><p>לְפָנָיו יִכְרְעוּ אַנְשֵׁי תְבוּנָה / וְתַחְתָּיו כָּל מְתֵי חָכְמָההֲדוּכִים</p></blockquote>
<blockquote><p>וְהוּקַם עָל לְהוֹרוֹת מִשְׁפְּטֵי אֵל / וְהִצִּיב קַו אֱמֶת לִפְנֵיחֲשׁוּכִים</p></blockquote>
<blockquote><p>עֲטֶרֶת כָּל מְתֵי חָכְמָה וְנֵזֶר / מְתֵי מִשְׂרָה וְחֶמְדַּתהַנְּסִיכִים</p></blockquote>
<blockquote><p>אֲדוֹן נַפְשִׁי, רְאֵה כִּי יִחֲלוּ לָךְ / וְהִנָּם נוֹשְׂאִי נַפְשָׁםוְחוֹכִים</p></blockquote>
<blockquote><p>לְפָנֶיךָ לְלַקֵּט [בֵּין עֳמָרִים] / לְבַל יִהְיוּ סְבוּכִים</p></blockquote>
<blockquote><p>וּבֵין הָעוֹמְדִים לִדְרֹשׁ תְּבוּנָה / יְשַׁו לָהֶם לְפָנָיו מַהֲלָכִים</p></blockquote>
<blockquote><p>בְּנָשׁוֹךְ נַחֲשֵׁי סִכְלוּת תְּשַׁו נֵס / תְּבוּנָתָךְ לְהַחְיוֹתהַנְּשׁוּכִים</p></blockquote>
<blockquote><p>וְיוֹם יוֹם תַּעֲרֹךְ נֵרוֹת תְּבוּנָה / וְאַל יִהְיוּ כְאֵשׁ קוֹצִיםדְּעוּכִים</p></blockquote>
<blockquote><p>וְיֵאוֹרוּ פְנֵיהֶם בַּהֲדָרָךְ / וּמִבִּרְכָתְךָ יִהְיוּ בְרוּכִים.</p></blockquote>
